Elektronik Grundlagen
=====================

Ohmsches Gesetz
---------------
`U = R * I` <=> `R = U / I` <=> `I = U / R`

LED-Vorwiderstand
-----------------
`R = U / I` => (*Source Voltage VS* - *Forward Voltage VF*) / *Forwand Current IF* = *Resistor R*

Beispiel: 5V Versorgungsspannung, VF=2V (rote LED) und IF=20mA => (5 - 2) / 0,02 = 150 Ohm Vorwiderstand
